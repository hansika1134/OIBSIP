# Calculator — OIBSIP Web Development Level 2, Task 1

A fully functional browser-based calculator built with vanilla HTML, CSS, and JavaScript.

## Features
- Numeric buttons (0–9) and decimal point
- Operators: addition, subtraction, multiplication, division
- Operator chaining (e.g., 5 + 3 × 2) without needing a reset
- Clear (C) and Backspace (⌫) buttons
- Division-by-zero handled gracefully with an error message (no crash)
- CSS Grid used for button layout
- All interactions wired with `addEventListener` — no inline `onclick`
- Keyboard support (digits, `+ - * /`, Enter, Backspace, Escape)

## Tech Stack
HTML5, CSS3, JavaScript (Vanilla) — no libraries, no `eval()`.

## How to Run
Open `index.html` in any modern browser. No build step or server required.

## Files
- `index.html` — structure
- `style.css` — styling and grid layout
- `script.js` — calculator logic and event handling
