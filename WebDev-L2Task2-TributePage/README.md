# Tribute Page — Ada Lovelace — OIBSIP Web Development Level 2, Task 2

A responsive tribute page dedicated to Ada Lovelace, regarded as the first computer programmer.

## Features
- Title with subject name and tagline
- Prominent portrait image (Wikimedia Commons)
- 4-paragraph original biography section (paraphrased from public sources)
- Timeline of key milestones (styled cards)
- Distinct quote block
- Two background colors (cream body / deep purple hero & quote sections)
- Two font families: Playfair Display (headings) + Source Sans Pro (body)
- Fully responsive layout, with a small scroll-reveal enhancement in JS

## Tech Stack
HTML5, CSS3, minimal vanilla JavaScript (progressive enhancement only).

## How to Run
Open `index.html` in a browser (requires internet connection to load the Google Fonts and Wikimedia image).

## Sources
Biographical facts paraphrased from Wikipedia and Britannica entries on Ada Lovelace. Portrait sourced from Wikimedia Commons (public domain).
